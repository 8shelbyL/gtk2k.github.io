#A-Frame stereoscopic media source support demo pages.
Stereoscopic type  
OU: Left eye:Over  / Right eye:Under.  
UO: Left eye:Under / Right eye:Over.  
LR: Left eye:Left  / Right eye:Right.  
RL: Left eye:Right / Right eye:Left.
